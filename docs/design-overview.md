# Design Overview
This is a web application that allows employees to submit feedback toward each other's performance review.

### Data
- employees
- reviews
- feedback
- feedback assignments

### Pages
#### Admin View
1. Employee List
2. Employee Detail
  - review list
3. Review Detail
  - feedback list

#### Employee View
4. Review List
5. Review Detail
  - feedback list

- Authentication TBD

### Web app
